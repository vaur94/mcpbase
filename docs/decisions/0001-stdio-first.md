# ADR 0001 - Stdio-first tasarim

## Karar

Bu temel depo yalnizca stdio transport ile gelir.

## Gerekce

- kapsam kontrollu kalir
- yerel host entegrasyonlari icin en dogrudan baslangic budur
- transport cesitliligi temel depoda gereksiz karmasa yaratir

## Sonuc

HTTP ve uzak dagitim turetilmis depolara birakilir.
