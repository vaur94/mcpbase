# Support

English | [Turkce](./SUPPORT.tr.md)

## Where to ask for help

- Bugs: open a GitHub issue using the bug report template.
- Feature requests: open a GitHub issue using the feature request template.
- Documentation fixes: open a pull request and mention the affected English and Turkish pages.
- Security issues: follow [`SECURITY.md`](./SECURITY.md) and avoid public disclosure before maintainer review.

## What to include

- Package version and Node.js version
- Relevant command output or reproduction steps
- Whether the problem affects stdio, Streamable HTTP, hub helpers, security guards, or docs only

## Manual confirmation items

The repository does not document a dedicated maintainer support email, chat channel, or SLA. If you need a private contact path outside GitHub, maintainer confirmation is required.
