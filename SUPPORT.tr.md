## Yardim isteme yollari

- Hata bildirimleri: bug report sablonu ile GitHub issue acin.
- Ozellik talepleri: feature request sablonu ile GitHub issue acin.
- Dokumantasyon duzeltmeleri: Etkilenen Turkce ve English sayfalari belirterek pull request acin.
- Guvenlik konulari: [`SECURITY.tr.md`](./SECURITY.tr.md) adimlarini izleyin ve maintainer incelemesinden once herkese acik paylasim yapmayin.

## Eklenmesi gereken bilgiler

- Paket surumu ve Node.js surumu
- Ilgili komut ciktilari veya tekrar uretim adimlari
- Sorunun stdio, Streamable HTTP, hub yardimcilari, security guard'lari veya yalnizca dokumantasyon ile ilgili olup olmadigi

## Elle dogrulama gereken noktalar
